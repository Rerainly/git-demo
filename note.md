初始化文件：git init
查看文件情况：git status
查看文件列表：ll
添加工作区文件：vim 文件名.后缀名
把工作区文件添加到缓存区：git add 文件
把缓存区文件删除：rm --cached 文件名

查看文件信息：cat 文件名

提交文件：git commit -m "注释信息" 文件名

查看版本：git reflog

查看详细版本：git log

穿越版本号，就是回到某个版本:git reset --hard 版本号
